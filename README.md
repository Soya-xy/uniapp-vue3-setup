
# uniapp app项目
 
## 项目介绍

集成 `tmui` `unplugin` `antfu-eslint`  一体的 uniapp 初始框架
封装路由跳转，请求，缓存，工具类等


## 注意事项

unocss@0.46.5, h5打包后 rpx 未转换成 rem

原因 unocss@0.46.5, 打包时 applyCssTransform 被删除了，导致打包后 rpx 未转换成 rem

解决方法: 先使用 0.46.4 版本

npm i unocss@0.46.4
